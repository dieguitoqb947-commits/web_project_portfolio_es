# Portafolio

## Nombre del proyecto

web_project_portfolio_es - Sprint 4

## Descripcion del proyecto y funcionalidad

Este proyecto es un portafolio personal creado para presentar mi perfil profesional, habilidades y proyectos destacados. Incluye una seccion principal con informacion de presentacion, un bloque de habilidades, una galeria de proyectos con enlaces externos y una seccion de contacto con redes sociales.

## Tecnologias y tecnicas utilizadas

- HTML5 con estructura semantica
- CSS3 con metodologia BEM
- Tipografias locales con `@font-face`
- Flexbox para maquetacion y distribucion
- Diseño responsivo con media queries
